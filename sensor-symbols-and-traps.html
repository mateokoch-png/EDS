Sensor families, circuit symbols & exam traps

So you stop mixing up which sensor belongs to which family — and recognize them when they show up in a diagram.

1. Current sensor family tree (CT ≠ Hall!) 2. Current sensor circuit symbols 3. Temperature sensor circuit symbols 4. Position sensor circuit symbols 5. Which sensors work with DC, and why 6. Exam trap library

1. Current sensor family tree

These are not all the same thing. They use three completely different physical principles:

Current sensors │ ├── RESISTIVE (Ohm's law: V = I·R) │ └── Shunt resistor │ ├── INDUCTIVE / TRANSFORMER (Faraday's law: needs CHANGING flux) │ └── Current Transformer (CT) ← AC ONLY │ └── MAGNETIC FIELD SENSING (reacts to the field's strength directly, works fine on steady/DC fields too) ├── Hall-effect family │ ├── Open-loop Hall sensor │ └── Closed-loop / Zero-flux Hall sensor (Hall element + feedback loop) │ └── Fluxgate sensor (saturable-core + AC excitation, NOT a Hall element)

The mix-up to avoid: a Current Transformer (CT) is in the inductive/transformer family — it has nothing to do with the Hall effect. "Zero-flux" / "closed-loop" sensors ARE built around a Hall element (plus a feedback loop) — so closed-loop sensors ARE part of the Hall family. Fluxgate is its own separate thing — it uses core saturation, not the Hall effect at all.

2. Current sensor circuit symbols

Shunt resistor
The zigzag = a normal resistor symbol. Blue wires on top = the separate Kelvin sense connection, going to a voltmeter.
Current Transformer (CT)
The circle = the donut-shaped core. The wire just passes straight through (1 turn). The wound coil at the bottom is the secondary, feeding a burden resistor.
Hall-effect current sensor (open-loop)
A small rectangular "Hall" chip symbol sits beside (or around) the conductor, with two extra output pins for V_out — no direct electrical connection to the power wire.
Closed-loop / zero-flux Hall sensor
Same core + Hall arrangement, but now with an extra feedback winding and a small amplifier box drawn alongside — this extra loop is the visual giveaway it's closed-loop, not open-loop.
Fluxgate sensor
An oval/racetrack-shaped core (so it saturates predictably), with an excitation winding drawn directly on it — plus, in full diagrams, a separate pickup winding and oscillator block.

3. Temperature sensor circuit symbols

Thermocouple
Two different-colored wires meeting at a single point (the junction) — always drawn as exactly two dissimilar conductors joined, never more.
RTD (e.g. Pt100)
A plain resistor box, just labeled "RTD" or "Pt100" — the symbol itself doesn't show anything fancy, only the label tells you it's temperature-sensitive.
NTC / PTC thermistor
A resistor box with a diagonal arrow through it (like a variable resistor) plus a small "t°" label — this diagonal-arrow-through-a-box is the standard symbol for any temperature-dependent resistor (NTC or PTC; the label/datasheet tells you which).

4. Position sensor (encoder) circuit symbols

Incremental encoder (block symbol)
A box labeled ENC with a shaft going in on one side, and three labeled output lines (A, B, Z) coming out the other side.
Absolute encoder (block symbol)
Same idea as incremental, but instead of separate A/B/Z lines, the output is usually a bundled multi-bit bus or a single serial interface (e.g. SSI) carrying the position code.

5. Which sensors work with DC, and why

SensorWorks on DC?Why
Shunt resistorYesOhm's law V=IR works regardless of whether I is AC or DC.
Current Transformer (CT)NoRelies on Faraday's law: needs a CHANGING flux to induce a voltage. Steady DC = steady flux = zero induced voltage.
Hall-effect (open or closed-loop)YesThe Hall element reacts to the field's instantaneous STRENGTH directly, not to a change in it. A steady DC field still produces a steady Hall voltage.
FluxgateYesIt generates its OWN internal AC excitation to probe the field — the field being measured doesn't need to change at all; only the internally-generated excitation needs to be AC.

The exam-ready one-liner: "CTs can't measure DC because they rely on induction (a changing flux), but Hall-effect and fluxgate sensors can, because they sense the magnetic field's strength directly rather than relying on a change in it."

6. Exam trap library

CT trap

"A current transformer can measure DC current." → False. CTs need changing flux (Faraday's law); steady DC produces zero induced secondary voltage.

Zero-flux trap

"In a zero-flux sensor, N₁·I₁ < N₂·I₂ at balance." → False. The feedback loop forces the flux to exactly zero, which requires an exact EQUALITY: N₁·I₁ = N₂·I₂.

PTC trap

"A PTC thermistor accurately measures temperature." → False. PTC is a binary protection trip (sharp resistance jump at threshold), not a precision continuous sensor.

Thermocouple trap

"A thermocouple generates a current proportional to temperature." → False (watch this swap — past exams alternated "voltage"/"current" between versions). A thermocouple generates a VOLTAGE (Seebeck effect); current only flows if you connect it in a closed loop with some resistance.

Encoder direction trap

"You need the Z (index) channel to detect direction of rotation." → False. Direction comes purely from which of A/B changes first. Z only gives a once-per-revolution reference position, unrelated to direction.

Encoder resolution trap

"A 50 PPR encoder gives 50 counts/rev." → False (unless explicitly told single-edge counting is used). With standard ×4 quadrature decoding, it's 50 × 4 = 200 counts/rev. Watch for whether the question specifies ×1, ×2, or ×4 decoding.

Shunt resistor trap

"You can measure the shunt voltage using the same two wires that carry the load current." → False (for high-accuracy applications). Lead/contact resistance in those wires adds error — a 4-wire Kelvin connection with separate sense leads is needed for precision.

RTD trap

"An RTD directly outputs a voltage proportional to temperature." → False. An RTD's resistance changes with temperature; you must pass a known excitation current through it and measure the resulting voltage drop to back-calculate resistance, then temperature.

Naming trap

"A current transformer is a type of Hall-effect sensor." → False. CT = inductive/transformer principle (Faraday's law). Hall sensors (open- or closed-loop) = semiconductor Hall effect. Two unrelated physical principles — don't conflate them.
